HTTP And HTTPS Protocol

  • HTTP or Hypertext Transfer Protocol is what is used between servers and browsers to talk to each other
  • This language is useful to exchange information such as HTML code
  • Your browser typically sends GET requests to obtain info like HTML code, images, or text
  • Occasionally your browser sends POST requests which give information like credit card details

  • Without HTTP we wouldn't be able to view or access websites
